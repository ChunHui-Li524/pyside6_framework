# 事件系统实现计划

## 1. 实现ExampleEvent类
在`example_event`文件中创建一个继承自QObject的ExampleEvent类，定义start和end信号。

```python
from PySide6.QtCore import QObject, Signal

class ExampleEvent(QObject):
    start = Signal()
    end = Signal()
```

## 2. 实现EventBus事件总线
在`__init__.py`文件中创建EventBus类，用于管理事件的发布和订阅。

```python
from PySide6.QtCore import QObject
from .example_event import ExampleEvent

class EventBus(QObject):
    def __init__(self):
        super().__init__()
        self._events = {}
        self._register_events()
    
    def _register_events(self):
        # 注册所有事件类
        self._events['example'] = ExampleEvent()
    
    def get_event(self, event_name):
        return self._events.get(event_name)
    
    # 快捷方法
    @property
    def example(self):
        return self.get_event('example')

# 实例化全局事件总线
event_bus = EventBus()
```

## 3. 使用方式
其他模块可以通过以下方式使用事件总线：

```python
from core.events import event_bus

# 订阅事件
event_bus.example.start.connect(self.on_example_start)
event_bus.example.end.connect(self.on_example_end)

# 发布事件
event_bus.example.start.emit()
event_bus.example.end.emit()
```

## 4. 扩展机制
当需要添加新的事件类时，只需：
1. 创建新的事件类文件
2. 在`__init__.py`的`_register_events`方法中注册新事件

这样设计可以确保事件系统的可扩展性和易用性，同时保持代码的模块化。
